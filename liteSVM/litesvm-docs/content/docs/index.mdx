## What is LiteSVM?

LiteSVM is an **in-process Solana VM** designed specifically for testing. Unlike traditional test validators that run as separate processes, LiteSVM runs directly within your test suite.

## Basic Example

Here's a simple test that demonstrates LiteSVM's power:

```rust title="tests/basic_test.rs"
use litesvm::LiteSVM;
use solana_sdk::{
    signature::{Keypair, Signer},
    system_instruction,
    transaction::Transaction,
};

#[test]
fn test_transfer() {
    // Create a new SVM instance
    let mut svm = LiteSVM::new();
    
    // Create test accounts
    let sender = Keypair::new();
    let receiver = Keypair::new();
    
    // Fund sender account
    svm.airdrop(&sender.pubkey(), 1_000_000_000).unwrap();
    
    // Create transfer transaction
    let instruction = system_instruction::transfer(
        &sender.pubkey(),
        &receiver.pubkey(),
        500_000_000,
    );
    
    let tx = Transaction::new_signed_with_payer(
        &[instruction],
        Some(&sender.pubkey()),
        &[&sender],
        svm.latest_blockhash(),
    );
    
    // Send transaction - executes immediately!
    let result = svm.send_transaction(tx).unwrap();
    assert!(result.is_ok());
    
    // Check balances - no waiting needed!
    assert_eq!(svm.get_balance(&sender.pubkey()).unwrap(), 500_000_000);
    assert_eq!(svm.get_balance(&receiver.pubkey()).unwrap(), 500_000_000);
}
```

<Callout>
  Notice how there's no `await` or sleep statements - everything executes synchronously!
</Callout>
